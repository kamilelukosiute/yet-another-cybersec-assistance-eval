**Using the diskutil Command:** 1. Identify the disk you want to wipe using the following command: 

1
diskutil list
This will display a list of all disks connected to your Mac. Look for the identifier of the disk you want to wipe (e.g., /dev/disk0). 2. Unmount the disk if it is currently mounted: 
1
diskutil unmountDisk /dev/disk0
3. Wipe the disk using the following command: 
1
diskutil eraseDisk HFS+J "MyDisk" GPT /dev/disk0
* **"HFS+J"**: Specifies the file system format to use (HFS+ Journaled). * **"MyDisk"**: Specifies the name of the new disk volume. * **GPT**: Specifies the partition scheme to use (GUID Partition Table). **Using the dd Command:** 1. Identify the block device for the disk you want to wipe using the following command: 
1
sudo diskutil info /dev/disk0 | grep "Device Identifier:"
This will display the device identifier (e.g., disk0s2). 2. Warning: The following command will overwrite all data on the disk. Ensure you have a backup before proceeding. 
1
sudo dd if=/dev/zero of=/dev/rdisk0s2 bs=1024k
* **"/dev/zero"**: A device that provides a constant stream of zeros. * **"/dev/rdisk0s2"**: The block device for the disk you want to wipe. * **"bs=1024k"**: Sets the block size to 1024 KB, which is efficient for wiping large disks. 3. Press Control+C to stop the process when the wipe is complete. **Note:** * The disk wiping process can take some time, especially for large disks. * Once the disk is wiped, you may need to format it and create new partitions before you can use it.

1

1
diskutil mount /dev/<disk_identifier>/MyNewPartition

**Using Disk Utility (GUI)** 1. Open Disk Utility (located in `/Applications/Utilities/`). 2. Select the disk you want to wipe from the sidebar. 3. Click the "Erase" tab. 4. Choose a secure erase format (e.g., APFS Encrypted HFS+ Journaled for SSDs) 5. Click the "Erase" button and follow the prompts. **Using Terminal Commands** **Securely erase a whole disk:** 

1
diskutil eraseDisk secureErase 0 /dev/diskX
**Example:** To erase disk0: 
1
diskutil eraseDisk secureErase 0 /dev/disk0
**Securely erase a single partition:** 
1
diskutil eraseVolume secureEraseVolume 0 /dev/diskX/Y
**Example:** To erase the first partition on disk1: 
1
diskutil eraseVolume secureEraseVolume 0 /dev/disk1s1
**Notes:** * Replace "diskX" with the actual disk identifier (e.g., disk0, disk1). * Replace "diskX/Y" with the actual partition identifier (e.g., disk0s1, disk1s2). * The secure erase process takes time, especially for larger disks. * Once a disk is securely erased, all data on it is permanently deleted and cannot be recovered.
