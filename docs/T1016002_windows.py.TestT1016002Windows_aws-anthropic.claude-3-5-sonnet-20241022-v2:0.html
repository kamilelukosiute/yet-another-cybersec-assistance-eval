Initial Query

I'm signed into my Windows laptop but I forgot my Wi-fi password so I can't connect other devices. Can I use the command line to enumerate the Wi-fi profiles and passwords for each?

LLM Generation

Query

I'm signed into my Windows laptop but I forgot my Wi-fi password so I can't connect other devices. Can I use the command line to enumerate the Wi-fi profiles and passwords for each?

Output

Yes, you can use Command Prompt (cmd) to view saved Wi-Fi passwords on Windows. Here's how: 1. Open Command Prompt as Administrator 2. To see all saved Wi-Fi profiles: 

1
netsh wlan show profiles
3. To see the password for a specific profile: 
1
netsh wlan show profile name="NETWORK-NAME" key=clear
(Replace NETWORK-NAME with the Wi-Fi network name) The password will be listed under "Key Content" in the Security settings section. You can also use this one-line command to see all profiles with passwords: 
1
for /f "tokens=4*" %i in ('netsh wlan show profiles ^| find "Profile"') do @echo off > nul & netsh wlan show profiles %i key=clear | findstr "SSID Cipher Content" | find /v "Number" & echo.
