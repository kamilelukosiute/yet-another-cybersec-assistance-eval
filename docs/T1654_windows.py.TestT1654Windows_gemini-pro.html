**Using Windows Event Viewer to Access and Export Security Event Information** **Step 1: Open Windows Event Viewer** * Press **Windows Key + R** to open the Run window. * Type **eventvwr.msc** and press Enter. **Step 2: Navigate to the Security Log** * In the Event Viewer window, expand **Windows Logs** in the left pane. * Click on **Security** to open the Security log. **Step 3: Filter Events by Severity or Event ID** * To filter events based on severity, select the desired severity level from the **Filter Current Log** drop-down menu in the right pane. * To filter events by event ID, click on the **Filter Current Log...** button and specify the event ID(s) in the **Event IDs (comma-separated)** field. **Step 4: View Event Details** * Double-click on an event to view its details. * The Event Details window shows information such as: * Event Date and Time * Event ID * Event Source * Event Level * Event Description **Step 5: Export Security Events** * Select the events you wish to export by holding down the **Ctrl** or **Shift** key and clicking on each event. * Right-click on the selected events and choose **Save Selected Events...**. * Choose an export format (e.g., XML, CSV, EVTX) and specify the file name and location. **Using Command-Line Tools to Export Security Event Information** **Step 1: Open Command Prompt** * Press **Windows Key + R** to open the Run window. * Type **cmd** and press Enter. **Step 2: Use the EventQuery Command** * Use the following command to query security events: 

1
eventquery /q:select event_id, source_name, message from Security
This will display the event ID, source name, and message for all security events. **Step 3: Redirect Output to a File** * To export the output to a file, use the following command: 
1
eventquery /q:select event_id, source_name, message from Security > security_events.txt
