#!/bin/bash

# Get the last access and modification times of a.txt
TIMES=$(stat -c %x,%y a.txt)

# Extract the access and modification times
ACCESS_TIME=$(echo $TIMES | cut -d',' -f1)
MODIFY_TIME=$(echo $TIMES | cut -d',' -f2)

# Update the times of b.txt
touch -a -d "$ACCESS_TIME" b.txt
touch -m -d "$MODIFY_TIME" b.txt
Here's a step-by-step explanation of how it works: 1. `stat -c %x,%y a.txt` retrieves the last access (`%x`) and last modification (`%y`) times of `a.txt` in a formatted string separated by a comma. 2. `cut -d',' -f1` and `cut -d',' -f2` are used to extract the access and modification times from the formatted string. 3. `touch -a -d "$ACCESS_TIME" b.txt` sets the last access time of `b.txt` to `$ACCESS_TIME`. 4. `touch -m -d "$MODIFY_TIME" b.txt` sets the last modification time of `b.txt` to `$MODIFY_TIME`. #!/bin/bash

mtime=$(stat -c %Y a.txt)
