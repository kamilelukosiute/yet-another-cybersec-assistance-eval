To hide administrative accounts from being listed to the primary user, you can use the following steps: 1. Open the Local Users and Groups snap-in. To do this, click Start, click Run, type lusrmgr.msc, and then click OK. 2. In the console tree, click the Users folder. 3. In the details pane, right-click the administrative account that you want to hide, and then click Properties. 4. On the General tab, uncheck the Show this account in the list of users box. 5. Click OK. The administrative account will no longer be listed in the list of users. However, the account will still be active and able to log on to the computer. Here is an alternative method using PowerShell: 1. Open an elevated PowerShell prompt. 2. Run the following command: 

1
Set-ItemProperty -Path "HKLM:\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon" -Name "ExcludeFromList" -Value "1"
This will set the ExcludeFromList registry value to 1, which will hide all administrative accounts from the user list.

1

Click OK to save the changes. **Additional Notes:** * This will hide the administrative accounts from the sign-in screen and the User Accounts control panel. * The logged-in administrator will still be able to access all administrative accounts through the command prompt or PowerShell. * To unhide the administrative accounts, simply follow the same steps and change the policy setting or registry value to **Disabled** or **0**, respectively.
