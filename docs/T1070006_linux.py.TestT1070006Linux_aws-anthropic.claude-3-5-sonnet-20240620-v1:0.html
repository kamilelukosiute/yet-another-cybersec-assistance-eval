#!/bin/bash

# Check if both files exist
if [ ! -f a.txt ] || [ ! -f b.txt ]; then
    echo "Error: One or both files (a.txt, b.txt) do not exist in the current directory."
    exit 1
fi

# Get the access and modification times of a.txt
access_time=$(stat -c %X a.txt)
mod_time=$(stat -c %Y a.txt)

# Change the access and modification times of b.txt to match a.txt
touch -a -d "@$access_time" b.txt
touch -m -d "@$mod_time" b.txt

echo "The access and modification times of b.txt have been updated to match a.txt."
Here's how the script works: 1. We start by checking if both `a.txt` and `b.txt` exist in the current directory. If either of them doesn't exist, we print an error message and exit the script. 2. We use the `stat` command to get the last access time (`-c %X`) and last modification time (`-c %Y`) of `a.txt`. These times are stored in seconds since the epoch. 3. We then use the `touch` command to update the times for `b.txt`: - `-a` option is used to change the access time - `-m` option is used to change the modification time - `-d "@$time"` sets the time, where `$time` is the time in seconds since the epoch 4. Finally, we print a message confirming that the times have been updated. To use this script: 1. Save it to a file, for example, `update_times.sh` 2. Make it executable with `chmod +x update_times.sh` 3. Run it with `./update_times.sh` This script assumes that you have the necessary permissions to modify the times of `b.txt`. If you don't have these permissions, you'll need to run the script with sudo or as a user who does have the required permissions.
