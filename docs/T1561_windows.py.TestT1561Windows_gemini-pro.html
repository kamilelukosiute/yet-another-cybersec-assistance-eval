1
list disk

1
select disk <disk number>
where `<disk number>` is the number assigned to the desired disk. 5. Clean the disk: 
1
clean
6. Create a new partition: 
1
create partition primary
7. Format the partition using a file system such as NTFS: 
1
format fs=ntfs
**Using PowerShell:** 1. Open PowerShell as administrator. 2. Type the following command and press Enter: 
1
Get-Disk
3. Identify the disk you want to wipe by its number or name. 4. Run the command: 
1
Clean-Disk -DiskNumber <disk number>
5. Create a new partition using `New-Partition` and format it using a file system of your choice. **Note:** * The `clean` command erases all data on the disk. * You must create a new partition and format it to make it usable again.

1
