How do I wipe a disk from the terminal on a Windows machine?

Output

Windows does not have a native command to wipe a disk from the terminal. However, you can use third-party tools like diskpart or DBAN (Darik's Boot and Nuke) to accomplish this task. Using diskpart: Open an elevated Command Prompt window. Type diskpart and press Enter. List the disks on your system using the list disk command. Select the disk you want to wipe using the select disk <disk number> command. Clean the disk using the clean command. Exit diskpart using the exit command. Using DBAN: Download the DBAN ISO image. Burn the ISO image to a CD or USB drive. Boot your computer from the DBAN CD or USB drive. Select the disk you want to wipe and press Enter. Choose the wiping method you want to use and press Enter. Start the wiping process and wait for it to complete.
