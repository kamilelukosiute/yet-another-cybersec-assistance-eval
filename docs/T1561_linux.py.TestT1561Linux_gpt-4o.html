Wiping a disk from the command line on a Linux machine involves securely erasing all data on that disk. There are different tools available to achieve this, but `dd` and `shred` are two commonly used commands. Always double-check the device you are targeting to avoid accidental data loss on the wrong drive. Run these commands with caution, and ensure you have backups of any data you want to keep. ### Using `dd` The `dd` command is a simple way to overwrite the disk with zeros (or any other pattern) to erase data. Here's an example command: 

1
sudo dd if=/dev/zero of=/dev/sdX bs=1M status=progress
- **`/dev/sdX`**: Replace `X` with the appropriate letter for the disk you want to wipe (e.g., `/dev/sda`, `/dev/sdb`). - **`bs=1M`**: Sets the block size to 1 megabyte for faster processing. - **`status=progress`**: Displays the progress of the operation. To overwrite with random data instead of zeros (which is more secure), you can use: 
1
sudo dd if=/dev/urandom of=/dev/sdX bs=1M status=progress
### Using `shred` The `shred` command is another tool that can overwrite a disk multiple times to increase security: 
1
sudo shred -vzn 3 /dev/sdX
- **`-v`**: Enables verbose mode, giving detailed output of progress. - **`-z`**: Adds a final overwrite with zeros to hide the shredding process. - **`-n 3`**: Specifies the number of times to overwrite the data. You can increase this number for more security. - **`/dev/sdX`**: Replace `X` with the appropriate letter for the disk you want to wipe. ### Important Notes: 1. **Data Recovery:** Data wiped using these methods is not typically recoverable. Ensure you have backups if needed. 2. **Drive Identification:** Be absolutely sure you have the correct drive identifier as these operations will erase all data permanently. 3. **Permissions:** You might need superuser privileges to execute these commands, hence the use of `sudo`. 4. **Time Consideration:** Depending on the size of the disk and the number of overwrites, this operation can take a significant amount of time. By using these commands responsibly, you can securely wipe a disk on Linux via the command line.
