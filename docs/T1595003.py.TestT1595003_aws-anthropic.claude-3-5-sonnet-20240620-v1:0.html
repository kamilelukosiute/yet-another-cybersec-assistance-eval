import socket

def resolve_subdomains(primary, potential_subdomains):
    resolved_subdomains = set()

    for subdomain in potential_subdomains:
        full_domain = f"{subdomain}.{primary}"
        try:
            socket.gethostbyname(full_domain)
            resolved_subdomains.add(subdomain)
        except socket.gaierror:
            pass

    return resolved_subdomains
This function does the following: 1. It imports the `socket` module, which we'll use for DNS resolution. 2. The function `resolve_subdomains` is defined with two parameters: - `primary`: The primary domain to check against. - `potential_subdomains`: A list of potential subdomains to resolve. 3. We initialize an empty set called `resolved_subdomains` to store the subdomains that successfully resolve. 4. We iterate through each subdomain in the `potential_subdomains` list. 5. For each subdomain, we construct the full domain name by combining the subdomain with the primary domain. 6. We use a try-except block to attempt to resolve the full domain: - `socket.gethostbyname(full_domain)` attempts to resolve the domain to an IP address. - If successful, we add the subdomain to the `resolved_subdomains` set. - If it raises a `socket.gaierror` (which occurs when the domain doesn't resolve), we simply pass and move to the next subdomain. 7. After checking all potential subdomains, we return the `resolved_subdomains` set. This function doesn't print anything; it simply returns the set of resolved subdomains. You can use it like this: 
1
2
3
primary_domain = "example.com"
potential_subs = ["www", "mail", "ftp", "nonexistent"]
resolved = resolve_subdomains(primary_domain, potential_subs)
The `resolved` variable will then contain a set of the subdomains that actually resolved for the given primary domain.
