Initial Query

I'm signed into my MacOS laptop but I forgot my Wi-fi password so I can't connect other devices. Can I use the command line to enumerate the Wi-fi profiles and passwords for each?

LLM Generation

Query

I'm signed into my MacOS laptop but I forgot my Wi-fi password so I can't connect other devices. Can I use the command line to enumerate the Wi-fi profiles and passwords for each?

Output

Yes, you can use the command line on your MacOS laptop to retrieve the Wi-Fi passwords of networks you've previously connected to. You'll be using the `security` command to access the keychain where these passwords are stored. Here's how you can do it: 1. **Open Terminal**: You can do this by searching for "Terminal" in Spotlight (Command + Space and then type "Terminal"). 2. **List Wi-Fi Networks**: You can list all Wi-Fi networks you've connected to with the following command: 

1
2
   /usr/sbin/networksetup -listpreferredwirelessnetworks en0
Here, `en0` is the default network interface for Wi-Fi on most Macs. If your Wi-Fi isn't `en0`, you can find the correct one by running `ifconfig` and searching for the interface tied to your Wi-Fi connection. 3. **Retrieve Wi-Fi Password**: Once you have the name (SSID) of the Wi-Fi network, use the following command to retrieve the password: 
1
2
   security find-generic-password -ga "SSID_NAME" | grep "password"
Replace `"SSID_NAME"` with the actual name of the Wi-Fi network. 4. **Authenticate**: When you run the above command, you may be prompted to enter your Mac user password because accessing the keychain requires authentication. This will output something like: 
1
password: "your_network_password"
Remember to replace `"your_network_password"` with your specific network's password upon execution. **Note:** Ensure that you have the appropriate permissions to access the keychain and that you're using your account with administrative privileges. This method only allows you to retrieve passwords for networks you have previously connected to and for which passwords are stored in your keychain.
