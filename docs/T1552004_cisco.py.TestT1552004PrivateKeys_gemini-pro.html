**CLI Command:** 

1
copy mvpnkey
**Parameters:** * **remote://host/foldername/** - The remote server hostname or IP address and the directory path to which the files will be copied. * **access-list name** - (Optional) The name of the access list you want to apply to the encryption of the exported keys. * **labelled** - (Optional) Use this parameter to keep the label of the exported key files on the remote server. * **mvpnkey-name mvpnkey** - The name of the MVPN key that you want to export. * **start mvpnkey-name start** - The name of the first MVPN key to be exported from the series. * **end mvpnkey-name end** - The name of the last MVPN key to be exported from the series. * **file-format** - (Optional) File format to which exported keys are to be saved. The default file format is binary on the local storage device, and is PEM when copied to a remote host. * `binary` - Keys are exported in binary format. * `pem` - Keys are exported in PEM format. **Example:** To export MVPN key named "key1" and save it to a remote host: 
1
copy mvpnkey remote://host/foldername/ key1
To export a series of MVPN keys starting with "key1" and ending with "key5" with PEM format: 
1
copy mvpnkey file-format pem remote://host/foldername/ start key1 end key5
