To change the modify, access, and create times of a file in PowerShell, you can use the `Set-ItemProperty` cmdlet to modify these attributes independently. First, you'll need to create the `System.IO.FileInfo` object for the file you want to modify. Then, you can set the `CreationTime`, `LastAccessTime`, and `LastWriteTime` properties. Here's an example command:

$filePath = "C:\path\to\your\file.txt"
$newDateTime = Get-Date "2023-10-14 15:30:00"

# Get the file info
$file = Get-Item $filePath

# Set the creation, access, and write times
$file.CreationTime = $newDateTime
$file.LastAccessTime = $newDateTime
$file.LastWriteTime = $newDateTime
Make sure to replace `"C:\path\to\your\file.txt"` with the actual file path, and set `$newDateTime` to your desired date and time. Adjust the date string in `Get-Date` according to your requirements.
