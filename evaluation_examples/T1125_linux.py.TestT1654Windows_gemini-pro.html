1

1
ffmpeg -f video4linux2 -video_size 640x480 -framerate 30 -i /dev/video0 webcam_feed.mp4
**Explanation:** * `-f video4linux2`: Specifies that we are using the Video for Linux Two (V4L2) API to capture video. * `-video_size 640x480`: Sets the video resolution to 640x480 pixels. * `-framerate 30`: Sets the frame rate to 30 frames per second. * `-i /dev/video0`: Specifies the device path of the webcam. * `webcam_feed.mp4`: Specifies the output file name and format (MP4). **4. Play the video feed:** Once the capture is complete, you can play the video feed using any media player on your Linux computer. For example: 
1
mplayer webcam_feed.mp4
**5. Stop capturing the video feed:** Press `Ctrl + C` to stop capturing the video feed. **Additional Options:** * To continuously capture the video feed, use the `-t` option in `ffmpeg`: 
1
ffmpeg -f video4linux2 ... -t 600 webcam_feed.mp4
* To preview the video feed live, use the `-vcodec copy` option in `ffmpeg`: 
1
ffmpeg -f video4linux2 ... -vcodec copy -an -

1
ffmpeg -f video4linux2 -i /dev/video0 -s 1280x720 -r 30 -vcodec h264 -b:v 500k udp://localhost:1234
**Accessing the Stream:** Once the stream is running, you can access it using a media player that supports UDP streaming (e.g., VLC, MPV). In VLC, open the "Open Media" dialogue and enter the following address:
