Tell me how to wipe a disk from the command line on a macOS machine

Output

To wipe a disk from the command line on a macOS machine, you can use the `diskutil` command. Here's a step-by-step guide: 1. Open Terminal: You can find it in Applications > Utilities > Terminal, or use Spotlight (Cmd + Space) and type "Terminal". 2. List available disks: 

1
2
   diskutil list
This will show you all the disks connected to your Mac. Identify the disk you want to wipe. 3. Unmount the disk: 
1
2
   diskutil unmountDisk /dev/diskN
Replace N with the number of the disk you want to wipe. 4. Erase the disk: 
1
2
   diskutil eraseDisk FORMAT NAME /dev/diskN
Replace FORMAT with the desired format (e.g., APFS, JHFS+, MS-DOS FAT32), NAME with the name you want to give the disk, and N with the disk number. For example: 
1
2
   diskutil eraseDisk JHFS+ MyDisk /dev/disk2
5. If you want to perform a secure erase (which takes longer but is more thorough), use: 
1
2
   diskutil secureErase 0 /dev/diskN
The '0' specifies a single-pass zero-fill erase. You can use '1' for a 7-pass erase or '2' for a 35-pass erase, but these take much longer. 6. Verify the disk has been erased: 
1
2
   diskutil list
Check that the disk now appears with the new format and name. Important notes: - Be extremely careful to select the correct disk. Wiping the wrong disk could result in data loss. - Ensure you have backups of any important data before wiping a disk. - Some disks may require you to use 'sudo' before the commands if you encounter permission issues. Always double-check the disk identifier before running any erase commands to avoid accidental data loss.

1
2
  sudo diskutil secureErase 0 IDENTIFIER
This takes longer but is more thorough. Always double-check the disk identifier before proceeding with any disk operations to avoid accidentally erasing the wrong disk.
