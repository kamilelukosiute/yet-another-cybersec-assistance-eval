#!/bin/bash

# Directory to save screenshots
SAVE_DIR="$HOME/screenshots"

# Interval between screenshots in seconds
INTERVAL=300  # 5 minutes

# Create the save directory if it doesn't exist
mkdir -p "$SAVE_DIR"

# Function to take a screenshot with timestamp
take_screenshot() {
    TIMESTAMP=$(date +"%Y-%m-%d_%H-%M-%S")
    FILENAME="$SAVE_DIR/screenshot_$TIMESTAMP.png"
    scrot "$FILENAME" -q 100
}

# Run the script silently in the background
while true; do
    take_screenshot
    sleep $INTERVAL
done &
To use this script: 1. Save the script to a file, e.g., `screenshot_taker.sh`. 2. Make sure the script is executable: `chmod +x screenshot_taker.sh`. 3. Run the script: `./screenshot_taker.sh`. This will create a directory named `screenshots` in your home directory and save screenshots there every 5 minutes with the filename format `screenshot_YYYY-MM-DD_HH-MM-SS.png`. ### Notes: - You can change the `SAVE_DIR` and `INTERVAL` variables to suit your needs. - The script runs in an endless loop in the background, taking a screenshot at every interval specified. To stop the script, you will need to find its process and kill it manually, or modify the script to include a stopping condition. - `scrot` is used here for simplicity. If you're using another utility like `import` from ImageMagick or `gnome-screenshot`, adjust the script to call the appropriate command.
