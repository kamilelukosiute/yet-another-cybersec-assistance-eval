Here are several methods to wipe a disk from the command line in Linux: 1. Using `dd` command (basic method): 

1
2
3
4
5
# Overwrite with zeros
dd if=/dev/zero of=/dev/sdX bs=4M status=progress

# Overwrite with random data
dd if=/dev/urandom of=/dev/sdX bs=4M status=progress
Replace `sdX` with your target drive (e.g., sda, sdb) 2. Using `shred` (more secure, multiple passes): 
1
shred -v -n 3 -z /dev/sdX
-v : verbose -n 3 : number of passes -z : add final pass with zeros 3. Using `wipefs` (to remove filesystem signatures): 
1
wipefs -a /dev/sdX
4. Using `hdparm` (for SSDs with secure erase support): 
1
2
hdparm --security-set-pass password /dev/sdX
hdparm --security-erase password /dev/sdX
IMPORTANT SAFETY NOTES: - Double-check the device name before running any commands - Make sure you're wiping the correct drive - These operations are irreversible - Never wipe a mounted drive - You need root privileges (sudo) for these commands To check your drive name: 
1
2
3
lsblk
# or
fdisk -l
