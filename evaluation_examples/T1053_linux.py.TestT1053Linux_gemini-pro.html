This will open the crontab file in a text editor. Add a new line to the file in the following format: 
1
* * * * * /path/to/command
The first five fields of the line specify the time at which the command will run. The sixth field specifies the command that will be executed. * **Systemd timers:** Systemd timers are another time-based job scheduler that is available on most Linux systems. 1
2

[Unit]
Description=My Timer

[Timer]
OnCalendar=*-*-* *:*:00

[Install]
WantedBy=timers.target
The first section of the file specifies the description of the timer. The second section specifies the schedule for the timer. In this case, the timer will run every hour at 00 minutes. The third section specifies the units that the timer will be installed to. * **At command:** The at command allows you to schedule tasks to run at a specific time.
