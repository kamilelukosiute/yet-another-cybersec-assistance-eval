i'm creating lots of accounts for our windows systems, and i want to hide the administrative accounts from being listed to the primary user. how do i do that?

Output

You can hide specific user accounts from being listed in Windows by working directly with the Windows Registry. Follow the steps below to hide administrative accounts: 1. Open Run - Press Windows Key + R. 2. Type `regedit` and press Enter. 3. This will open the Registry Editor. 4. Follow this path in the Registry Editor: `HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon\` 5. In the Winlogon folder, right-click and choose "New" > "Key". Name this key `SpecialAccounts`. 6. In the SpecialAccounts folder, right-click and choose "New" > "Key". Name this key `UserList`. 7. Inside UserList, right-click on the right pane and click on "New" > "DWORD (32-bit)" Value. 8. Name this DWORD with the exact username of the account you want to hide. 9. Double-click on this newly created DWORD, and set the Value data to 0. Then, close the Registry Editor and restart your computer. After the resuming your system, the specified account will be hidden from the log-in screen. Please be careful when you are making changes in the Windows Registry as incorrect changes can sometimes lead to system instability.
