Output

To wipe a disk from the terminal on a Windows machine, you can use the `diskpart` command-line utility. This tool allows you to manage disks, partitions, and volumes. Here's a step-by-step guide on how to wipe a disk using `diskpart`: **Warning**: This process will permanently erase all data on the selected disk. Make sure you have backed up any important data before proceeding. 1. **Open Command Prompt as Administrator**: - Type "cmd" in the Windows search bar. - Right-click on "Command Prompt" and select "Run as administrator". 2. **Start Diskpart**: - In the Command Prompt window, type `diskpart` and press Enter. This will open the Diskpart utility. 3. **List All Disks**: - Enter `list disk` and press Enter. This will display all disks connected to your computer. 4. **Select the Disk to Wipe**: - Identify the disk you want to wipe by its number. For example, if you want to wipe "Disk 1", type `select disk 1` and press Enter. 5. **Clean the Disk**: - Type `clean` and press Enter. This command will remove all partitions or volume formatting from the disk, effectively wiping it. 6. **(Optional) Perform an Advanced Wipe**: - If you want to thoroughly wipe the disk by writing zeros to every sector, type `clean all` instead of `clean`. Please note that this process will take longer. 7. **Exit Diskpart**: - After the process is complete, type `exit` and press Enter to leave Diskpart utility. 8. **Close Command Prompt**: - Type `exit` once more to close the Command Prompt window. After completing these steps, the selected disk will be wiped and will no longer contain any data or partitions. You can now choose to partition and format the disk as needed for further use.
