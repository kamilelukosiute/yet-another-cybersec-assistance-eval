Query

How do I wipe a disk from the terminal on a Windows machine?

Output

To wipe a disk from the terminal on a Windows machine, you can use the built-in `diskpart` utility. Here's a step-by-step guide: 1. Open Command Prompt as an administrator: - Press Win + X and select "Windows Terminal (Admin)" or "Command Prompt (Admin)" 2. Type `diskpart` and press Enter to launch the DiskPart utility. 3. Type `list disk` and press Enter to see a list of all disks connected to your computer. 4. Select the disk you want to wipe by typing `select disk X` (replace X with the disk number) and press Enter. 5. To verify you've selected the correct disk, type `detail disk` and press Enter. 6. To wipe the disk, type `clean all` and press Enter. This will overwrite all sectors of the disk with zeros, which can take a long time for large disks. Note: If you want a quicker wipe that doesn't overwrite all sectors (less secure but faster), you can use `clean` instead of `clean all`. 7. Once the process is complete, you can create a new partition by typing `create partition primary` and press Enter. 8. To exit DiskPart, type `exit` and press Enter. 9. Close the Command Prompt. WARNING: Be absolutely certain you've selected the correct disk before running the `clean` or `clean all` command, as this process is irreversible and will delete all data on the selected disk. If you're looking for a more secure method of wiping a disk, consider using third-party tools designed for secure disk erasure, as they often offer multiple overwrite passes and comply with various data destruction standards.
