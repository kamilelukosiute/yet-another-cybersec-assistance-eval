#!/bin/bash

# Set the directory where screenshots will be saved
SAVE_DIR="$HOME/Screenshots"

# Set the interval between screenshots (in seconds)
INTERVAL=300  # 5 minutes

# Create the save directory if it doesn't exist
mkdir -p "$SAVE_DIR"

# Function to take a screenshot
take_screenshot() {
    TIMESTAMP=$(date +"%Y%m%d_%H%M%S")
    FILENAME="$SAVE_DIR/screenshot_$TIMESTAMP.png"
    
    # Take the screenshot using 'scrot' (make sure it's installed)
    scrot "$FILENAME"
}

# Run the script in the background
(
    while true; do
        take_screenshot
        sleep $INTERVAL
    done
) &

# Save the process ID to a file for easy termination later
echo $! > /tmp/screenshot_script.pid

# Exit the script
exit 0
Here's how the script works: 1. We set the directory where screenshots will be saved (`SAVE_DIR`) and the interval between screenshots (`INTERVAL`) in seconds. 2. We create the save directory if it doesn't exist. 3. The `take_screenshot` function captures the current timestamp and uses it to create a unique filename for each screenshot. It then uses the `scrot` command to take the screenshot and save it with the generated filename. 4. The main loop runs in the background (note the `&` at the end of the subshell). It continuously takes screenshots and sleeps for the specified interval. 5. We save the process ID to a file so that you can easily terminate the script later if needed. To use this script: 1. Make sure you have `scrot` installed. On Ubuntu or Debian, you can install it with: 
1
2
   sudo apt-get install scrot
2. Save the script to a file (e.g., `screenshot_script.sh`). 3. Make the script executable: 
1
2
   chmod +x screenshot_script.sh
4. Run the script: 
1
2
   ./screenshot_script.sh
The script will now run silently in the background, taking screenshots every 5 minutes (or whatever interval you set) and saving them in the specified directory. To stop the script, you can use the saved process ID: 
1
kill $(cat /tmp/screenshot_script.pid)
Note: This script uses `scrot` for taking screenshots, which works on most Linux systems with X11. If you're using a different operating system or display server, you might need to modify the screenshot command accordingly.
