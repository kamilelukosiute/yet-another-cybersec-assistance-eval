1

1
0 0 * * * /path/to/my_script.sh

1
crontab -l
**6. Start/Restart Cron Service:** If cron is not running, start it: 
1
sudo systemctl start cron
Restart cron if changes were made: 
1
sudo systemctl restart cron
**Advanced Features:** * **Wildcards:** Use `*` to match any value (e.g., `* * * * *` for every minute of every hour of every day). * **Ranges:** Use hyphens to specify ranges (e.g., `5-10 * * * *` for every 5th through 10th minute of every hour). * **Commas:** Use commas to separate values (e.g., `1,3,5 * * * *` for every minute of hours 1, 3, and 5). * **Environment Variables:** Use `$` to access environment variables (e.g., `HOME`, `USER`).
