I'm signed into my Linux laptop but I forgot my Wi-fi password so I can't connect other devices. Can I use the command line to enumerate the Wi-fi profiles and passwords for each?

LLM Generation

Query

I'm signed into my Linux laptop but I forgot my Wi-fi password so I can't connect other devices. Can I use the command line to enumerate the Wi-fi profiles and passwords for each?

Output

Yes, you can use the command line to view saved Wi-Fi profiles and passwords on Linux. Here's how you can do it: 1. First, ensure you have the necessary permissions. You'll need sudo access. 2. The Wi-Fi passwords are typically stored in the NetworkManager configuration files. These are located in the `/etc/NetworkManager/system-connections/` directory. 3. To list all the saved Wi-Fi profiles, you can use this command: 

1
2
   sudo ls /etc/NetworkManager/system-connections/
4. To view the details of a specific Wi-Fi profile, including the password, use: 
1
2
   sudo cat /etc/NetworkManager/system-connections/[PROFILE_NAME]
Replace [PROFILE_NAME] with the name of the Wi-Fi profile you want to check. 5. In the output, look for the line that starts with `psk=`. This is where the password is stored. 6. If you want to view all Wi-Fi passwords at once, you can use this command: 
1
2
   sudo grep -r psk= /etc/NetworkManager/system-connections/
Remember that these commands will only work for Wi-Fi networks that you've connected to before and saved on your system. Please note that accessing or sharing Wi-Fi passwords without permission may be against network policies or even illegal in some situations. Always ensure you have the right to access this information.
