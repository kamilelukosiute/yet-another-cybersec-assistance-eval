Initial Query

I'm signed into my MacOS laptop but I forgot my Wi-fi password so I can't connect other devices. Can I use the command line to enumerate the Wi-fi profiles and passwords for each?

LLM Generation

Query

I'm signed into my MacOS laptop but I forgot my Wi-fi password so I can't connect other devices. Can I use the command line to enumerate the Wi-fi profiles and passwords for each?

Output

Yes, you can use the command line to enumerate the Wi-Fi profiles and passwords for each on your macOS laptop. Here's how: 1. Open the Terminal application on your Mac. 2. Type the following command and press Enter: 

1
security find-generic-password -ga WIFI | grep -i SSID
This command will list all the Wi-Fi profiles stored on your Mac, along with their SSIDs (network names). 3. To view the password for a specific Wi-Fi profile, type the following command and press Enter: 
1
security find-generic-password -a WIFI -s <SSID>
Replace `<SSID>` with the SSID of the Wi-Fi profile for which you want to view the password. The output of this command will include the password for the specified Wi-Fi profile. **Note:** You may be prompted to enter your macOS user password to view the passwords. Here is an example output of the commands:
