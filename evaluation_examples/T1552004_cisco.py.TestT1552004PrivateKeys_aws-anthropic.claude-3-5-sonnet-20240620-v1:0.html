How do I export keys on a cisco catalyst 9400 via the cli?

Output

To export keys on a Cisco Catalyst 9400 switch via the CLI, you'll need to follow these steps: 1. Connect to the switch using SSH or console. 2. Enter privileged EXEC mode: 

1
2
   enable
3. Generate a crypto key export command. The general syntax is: 
1
2
   crypto key export rsa [key-name] pem [usage-keys] terminal [passphrase]
Where: - [key-name] is the name of the key you want to export - [usage-keys] is optional and specifies whether to export usage keys - [passphrase] is an optional encryption passphrase 4. For example, to export a key named "mykey" with a passphrase: 
1
2
   crypto key export rsa mykey pem terminal mypassphrase
5. The switch will display the exported key in PEM format. Copy this output, including the "-----BEGIN RSA PRIVATE KEY-----" and "-----END RSA PRIVATE KEY-----" lines. 6. Save the copied key to a file on your local machine with a .pem extension. Remember: - You need to have the appropriate privileges to perform this operation. - Be cautious when exporting keys, as they are sensitive security information. - Always store exported keys securely. - If you're exporting for backup purposes, consider using a secure method to transfer and store the keys. Note that the exact command might vary slightly depending on your specific IOS version. If you encounter any issues, consult the Cisco documentation for your specific switch model and IOS version.
